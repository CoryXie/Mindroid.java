Contains high-level classes encapsulating the overall Mindroid application model.

An Mindroid application is defined using one or more of Mindroid's core application components calles {@link mindroid.app.Service}s.

A {@link mindroid.app.Service} is an application component that can perform long-running operations in the background. For example, a service can handle network transactions.
